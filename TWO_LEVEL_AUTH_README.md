# نظام المصادقة المطور - المستويين

## نظرة عامة
تم تطوير نظام مصادقة متقدم يدعم مستويين من التحقق في فورم التسجيل الموجود:

### المستوى العادي (Standard Level)
- تسجيل بالبيانات الأساسية
- تحقق بالبريد الإلكتروني
- تحقق برقم الهاتف (SMS)
- كلمة مرور آمنة
- حماية أساسية

### المستوى المتقدم (Advanced Level)
- تكامل كامل مع **World ID** للمصادقة البيومترية
- عمليات **KYC/AML** للتحقق من الهوية ومكافحة غسيل الأموال
- تكامل مع **Internet Identity** للـ ICP
- مصادقة متعددة العوامل (MFA)
- تحقق بيومتري متقدم
- حماية فائقة

## المكونات المضافة

### 1. خدمات التكامل
- `worldIdService.js` - تكامل World ID
- `internetIdentityService.js` - تكامل Internet Identity للـ ICP
- `kycAmlService.js` - عمليات KYC/AML

### 2. مكونات الواجهة
- `UserVerificationCard.jsx` - بطاقة اختيار مستوى التحقق
- تحديث `AuthPage.jsx` - دعم المستويين

## الميزات الجديدة

### اختيار مستوى التحقق
- واجهة تفاعلية لاختيار المستوى المناسب
- عرض مميزات كل مستوى بوضوح
- تصميم بطاقات جذاب مع أيقونات مميزة

### التحقق المتقدم
- مؤشرات حالة التحقق في الوقت الفعلي
- تكامل متوازي مع الخدمات المختلفة
- معالجة الأخطاء والاستثناءات

### تجربة المستخدم المحسنة
- انتقالات سلسة بين المراحل
- رسائل حالة واضحة
- تصميم متجاوب وجذاب

## كيفية الاستخدام

### للمستخدمين الجدد:
1. اختر نوع المستخدم (تاجر، عميل، سائق، شركة)
2. اختر مستوى التحقق (عادي أو متقدم)
3. أدخل البيانات المطلوبة
4. اتبع خطوات التحقق حسب المستوى المختار

### للمستخدمين الموجودين:
1. اختر نوع المستخدم
2. أدخل بيانات تسجيل الدخول
3. سيتم التحقق حسب مستوى الحساب المسجل

## التكاملات المطلوبة

```env
REACT_APP_KYC_BASE_URL=your_kyc_base_url
```

### المكتبات المطلوبة

- Internet Identity SDK
- مكتبات التشفير
